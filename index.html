About The Project

Company Internal Chatbot with RBAC

A secure internal chatbot system using Retrieval-Augmented Generation (RAG)

üéØ

Project Goal

Build a secure internal chatbot that processes natural language queries and retrieves department-specific company information. The system authenticates users, assigns roles (Finance, Marketing, HR, Engineering, C-Level, Employees), and provides role-based access to company documents stored in a vector database.

üìÑ

Document Processing

Extract, preprocess, and index company documents (markdown and CSV) into a vector database with role-based metadata tags.

üîê

Secure Authentication

Implement secure user authentication and role-based access control (RBAC) middleware for data protection.

RAG Pipeline

Build a RAG pipeline integrating semantic search with free LLMs to generate evidence-based responses.

Streamlit Interface

Develop a Streamlit web interface for user login, chat interaction, and role-specific information retrieval.
